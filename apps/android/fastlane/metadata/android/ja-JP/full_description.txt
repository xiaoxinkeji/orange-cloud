Orange Cloud は、Android スマートフォンとタブレット向けのネイティブな Cloudflare 管理クライアントです。Cloudflare 公式の OAuth でログインし、付与する権限を選んで、どこからでもドメインを管理できます。API トークンの作成・コピー・貼り付けは一切不要です。

【安全なログイン】
- 公式 OAuth 2.0 + PKCE：ブラウザで認証、ウェブ版と同じ流れ
- 権限は自分で選択。アプリは付与した分しか受け取りません
- トークンは Android Keystore に保存され、端末から外に出ません
- API リクエストは端末から Cloudflare へ直接。中間サーバーなし、データ収集なし

【ドメインと DNS】
- すべてのドメインを一覧で。ステータスとプランも一目で
- DNS レコードの追加・編集・削除・検索
- プロキシ（あのオレンジの雲）をワンタップで切り替え

【トラフィック分析】
- リクエスト数、帯域、キャッシュヒット率、ブロックした脅威
- すっきりしたグラフ：24 時間は無料、7 日・30 日は Pro

【Workers】
- デプロイ済みスクリプト、詳細、メトリクスを表示
- Pro：WebSocket でライブログを配信、常駐通知で同期

【ストレージ（Pro）】
- R2：バケットとオブジェクトの閲覧、アップロード・ダウンロード
- D1：内蔵 SQL コンソールでクエリ実行、テーブル行の閲覧・編集
- KV：キーの読み取り・書き込み・削除

【セキュリティとネットワーク（Pro）】
- WAF カスタムルール：その場で確認・作成・オン/オフ
- Cloudflare Tunnel：ステータスとアクティブな接続を確認

【Android のために】
- 時刻に合わせて移ろう昼夜テーマ、Material You のダイナミックカラー対応
- タブレットと折りたたみ向けの 2 ペインレイアウト
- ホーム画面ショートカット（アイコン長押し）とクイック設定タイル
- 予測型「戻る」、アプリごとの言語設定

【無料と Pro】
無料版はずっと使えます：1 アカウント、ドメインと DNS の完全管理、24 時間分析。
Orange Cloud Pro では、複数アカウントの切り替え、ストレージ全体（R2 / D1 / KV）、Workers のライブログ、WAF ルール、Tunnel、7 日・30 日分析が使えます。月額・年額サブスクリプション（年額は 7 日間の無料トライアル付き）または買い切りをご用意。

サブスクリプションは Google Play で課金され、現在の期間が終わる 24 時間以上前に解約しない限り自動更新されます。解約・管理は Google Play からいつでも可能です。
