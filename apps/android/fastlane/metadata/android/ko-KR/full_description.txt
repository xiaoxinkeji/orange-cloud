Orange Cloud는 Android 휴대폰과 태블릿을 위한 네이티브 Cloudflare 클라이언트입니다. Cloudflare의 공식 OAuth로 로그인하고, 부여할 권한을 선택한 뒤 어디서나 도메인을 관리하세요. 생성·복사·붙여넣을 API 토큰이 필요 없습니다.

안전하게 로그인
- PKCE를 사용하는 공식 OAuth 2.0: 웹과 똑같이 브라우저에서 인증
- 범위는 직접 선택; 앱은 부여한 것만 받습니다
- 토큰은 Android Keystore에 저장되며 기기를 절대 벗어나지 않습니다
- API 요청은 기기에서 Cloudflare로 바로 전송됩니다. 중간 서버 없음, 데이터 수집 없음

도메인 및 DNS
- 모든 도메인을 상태·요금제와 함께 한눈에
- 완전한 DNS 관리: 추가, 편집, 삭제, 검색
- 한 번 탭으로 프록시 전환(주황색 구름 자체)

분석
- 요청, 대역폭, 캐시 적중률, 차단된 위협
- 깔끔한 차트: 최근 24시간 무료, 7일·30일 범위는 Pro

WORKERS
- 배포된 스크립트, 세부 정보, 지표 탐색
- Pro: 지속 알림과 함께 WebSocket으로 실시간 로그 스트리밍

스토리지 (PRO)
- R2: 버킷과 객체 탐색, 업로드 및 다운로드
- D1: 내장 SQL 콘솔에서 쿼리 실행, 테이블 행 탐색 및 편집
- KV: 키 읽기, 쓰기, 삭제

보안 및 네트워크 (PRO)
- WAF 사용자 지정 규칙: 이동 중에 검토, 생성, 전환
- Cloudflare Tunnel: 상태와 활성 연결 확인

ANDROID를 위해 제작
- 시간에 따라 바뀌는 주야간 테마, Material You 동적 색상
- 태블릿과 폴더블을 위한 적응형 2열 레이아웃
- 홈 화면 바로가기(아이콘 길게 누르기)와 빠른 설정 타일
- 예측형 뒤로 가기 및 앱별 언어

무료 vs PRO
영원히 무료: 단일 계정, 전체 도메인 및 DNS 관리, 24시간 분석.
Orange Cloud Pro는 다중 계정, 전체 스토리지 탭(R2 / D1 / KV), 실시간 Workers 로그, WAF 규칙, Tunnel, 7일 / 30일 분석을 잠금 해제합니다. 월간 또는 연간 구독(연간은 7일 무료 체험 포함) 또는 일회성 평생 구매로 제공됩니다.

구독은 Google Play를 통해 청구되며 현재 기간 종료 최소 24시간 전에 취소하지 않으면 자동으로 갱신됩니다. Google Play에서 언제든지 관리하거나 취소하세요.
