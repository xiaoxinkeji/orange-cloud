Orange Cloud 是一款为 Android 手机与平板原生打造的 Cloudflare 管理客户端。使用 Cloudflare 官方 OAuth 授权登录，按需勾选权限，随时随地管理你的域名。无需创建、复制、粘贴任何 API Token。

【安全登录】
- 官方 OAuth 2.0 + PKCE 授权，在浏览器中完成，与网页端一致
- 权限自己选，App 只拿到你授予的部分
- Token 只存系统 Keystore，不离开设备
- API 请求由设备直连 Cloudflare，没有中间服务器，不收集你的数据

【域名与 DNS】
- 全部域名一眼看清，状态、套餐尽在掌握
- DNS 记录增删改查，支持搜索
- 代理（那朵小橙云）一键开关

【流量分析】
- 请求数、带宽、缓存命中率、拦截威胁
- 图表清爽：24 小时免费看，Pro 解锁 7 天 / 30 天

【Workers】
- 浏览已部署脚本、详情与指标
- Pro：WebSocket 实时日志直达，常驻通知同步

【存储（Pro）】
- R2：浏览存储桶与对象，上传与下载
- D1：内置 SQL 查询控制台，浏览与编辑表行
- KV：键值读、写、删

【安全与网络（Pro）】
- WAF 自定义规则：随时查看、新建与启停
- Cloudflare 隧道：查看状态与活跃连接

【为 Android 而生】
- 亮暗主题随晨昏自然切换，支持 Material You 动态取色
- 平板与折叠屏自适应双栏
- 桌面图标长按快捷、下拉快设磁贴
- 预测式返回手势、应用级语言设置

【免费与 Pro】
免费版永久可用：单账号、完整域名与 DNS 管理、24 小时分析。
Orange Cloud Pro 解锁多账号切换、整个存储模块（R2 / D1 / KV）、Workers 实时日志、WAF 规则、隧道，以及 7 天 / 30 天分析。提供月度 / 年度订阅（年度含 7 天免费试用）或一次性买断。
