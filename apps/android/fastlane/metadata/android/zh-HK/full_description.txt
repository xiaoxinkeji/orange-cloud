Orange Cloud 是一款為 Android 手機與平板原生打造的 Cloudflare 管理用戶端。以 Cloudflare 官方 OAuth 授權登入，按需勾選權限，隨時隨地管理你的域名。無需建立、複製、貼上任何 API Token。

【安全登入】
- 官方 OAuth 2.0 + PKCE 授權，在瀏覽器中完成，與網頁端一致
- 權限自己選，App 只拿到你授予的部分
- Token 只存系統 Keystore，不離開裝置
- API 請求由裝置直連 Cloudflare，沒有中間伺服器，不收集你的資料

【域名與 DNS】
- 全部域名一眼看清，狀態、計劃盡在掌握
- DNS 記錄增刪改查，支援搜尋
- 代理（那朵小橙雲）一鍵開關

【流量分析】
- 請求數、頻寬、緩存命中率、攔截威脅
- 圖表清爽：24 小時免費看，Pro 解鎖 7 天 / 30 天

【Workers】
- 瀏覽已部署指令碼、詳情與指標
- Pro：WebSocket 即時日誌直達，常駐通知同步

【儲存（Pro）】
- R2：瀏覽儲存桶與物件，上傳與下載
- D1：內置 SQL 查詢主控台，瀏覽與編輯資料列
- KV：鍵值讀、寫、刪

【安全與網絡（Pro）】
- WAF 自訂規則：隨時檢視、新增與啟停
- Cloudflare Tunnel：檢視狀態與活躍連線

【為 Android 而生】
- 亮暗主題隨晨昏自然切換，支援 Material You 動態取色
- 平板與摺疊螢幕自適應雙欄
- 桌面圖示長按捷徑、下拉快速設定圖塊
- 預測式返回手勢、應用程式語言設定

【免費與 Pro】
免費版永久可用：單帳戶、完整域名與 DNS 管理、24 小時分析。
Orange Cloud Pro 解鎖多帳戶切換、整個儲存模組（R2 / D1 / KV）、Workers 即時日誌、WAF 規則、Tunnel，以及 7 天 / 30 天分析。提供月度 / 年度訂閱（年度含 7 天免費試用）或一次性買斷。

訂閱透過 Google Play 計費，除非在目前訂閱期結束前至少 24 小時取消，否則將自動續訂。可隨時在 Google Play 管理或取消。
